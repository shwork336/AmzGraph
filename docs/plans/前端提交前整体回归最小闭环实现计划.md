# 前端提交前整体回归最小闭环实现计划

> 本计划承接前端路由按需拆包优化最小闭环。目标是在提交前对本轮前端工作台扩展做整体回归，确认构建、后端测试、路由注册、忽略文件和待提交集合都处于可提交状态。

## 1. 当前基线

已完成：

- 前端任务列表、创建任务、任务详情。
- 前端竞品、Brief、文案、图片、合规、终审、归档导出、操作审计页面。
- 前端顶部导航和任务详情工作流入口。

- 前端构建通过。

本阶段做：

- 命令级回归检查。

本阶段不做：

- 不新增业务页面。
- 不修改后端业务逻辑。
